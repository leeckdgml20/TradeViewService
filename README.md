# 거래 내역 조회 서비스 (TradeViewService)

### 버전
빌드 및 실행 환경
JDK 1.8
Maven 3

### 소스 내려받기
> $ git clone https://github.com/leeckdgml20/TradeViewService

### 설치 방법
* Root 이동
> $ cd TradeViewService

* 컴파일
> $ mvn compile

* 패키징
> $ mvn package

### 실행 방법
* WAS 실행
> $ mvn spring-boot:run
> $ java -jar target/TradeViewService-1.0.war

* 단위 테스트
> $ mvn test

* Client 접속
> http://localhost:8080/main
